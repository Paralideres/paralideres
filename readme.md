## Development
- Install [Composer](https://getcomposer.org/doc/00-intro.md)
- Install [Vagrant](https://www.vagrantup.com/downloads.html)
- Install [Virtual Box](https://www.virtualbox.org/wiki/Downloads)

To get the machine started do:
`composer install`
This will install all the PHP dependencies of the project.

To run the virtual machine for local development do:
`vagrant up`

To access the local development virtual machine do:
`vagrant ssh`
